## 🚀 How to Run It

1. Make sure you have Python installed (recommend Python 3.x).
2. Install Pygame:
   ```bash
   pip install pygame
   ```
3. Clone or download this repository:
   ```bash
   git clone https://github.com/your-username/pygame-game.git
   ```
4. Run the game:
   ```bash
   python main.py
   ```
   *(or replace `main.py` with your actual game file name)*
